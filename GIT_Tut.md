# Init the Git Repo:
```
git clone https://github.com/gbotkin3/ECE8803_Final_Project.git
```
# Switching to a New Branch
```
git checkout -b my-feature
```
# Pushing all Changes

Perform from the top-level directory

```
git add *
git commit -m "Commit Message"
git push
```

# Reverting Changes

```
git reset --hard
```
